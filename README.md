## 📦 Installation

If you'd like to run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/harsh-b-patel/Portfolio.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Portfolio
   ```
3. Open `index.html` in your browser.
